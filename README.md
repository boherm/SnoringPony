# SnoringPony

## Overview
SnoringPony is a juce-based open-source application designed for live theater productions to manage cues and sound effects seamlessly. It provides a user-friendly interface (based on [juce_organicui](https://github.com/benkuper/juce_organicui) from [Benjamin Kuperberg](https://benjamin.kuperberg.fr/)) for operators to trigger sound cues, OSC cues, manage playlists, etc... during performances.

For more details, visit the [official website](https://snoringpony.com) or the [documentation](https://docs.snoringpony.app)!

Feel free to join our [Discord community](https://discord.gg/7HNzrvuZUS) for support and discussions!

## How to test

**Download the latest release** from the [Releases page](https://snoringpony.app/#downloads) or build from source (see below).

## How to build

1. **Clone the Repository**:
   ```bash
   git clone --recurse-submodules
    ```
2. **Build via CMAKE (with Make command!)**:
   ```bash
   make
   ```
